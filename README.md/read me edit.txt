Configuración de Formspree
Para que el formulario de contacto funcione, utilicé Formspree, un servicio que permite procesar formularios de HTML puro sin necesidad de programar un servidor (Backend) propio.

Pasos realizados:

Creé una cuenta en formspree.io.

Generé un nuevo "Form" y obtuve el ID único.

Integré ese ID en el atributo action de la etiqueta <form> en mi HTML.

Definí el método como POST.

¿Por qué es útil?
Es una herramienta excelente para proyectos de frontend porque nos ahorra la complejidad de manejar lenguajes de servidor (como PHP o Node.js) y bases de datos. Además, Formspree ofrece protección básica contra spam y nos envía los mensajes directamente a nuestro correo electrónico.
