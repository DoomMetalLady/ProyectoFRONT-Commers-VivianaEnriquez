# ProyectoFRONT - Commers (Supermercado Online)

¡Bienvenido a mi primer proyecto web! Este sitio fue desarrollado como una base para un proyecto , enfocado en maquetación funcional, estructura limpia y diseño adaptable.

## 🛒 Sobre el Proyecto Unico
Es una simulación de un **Supermercado Online** (E-commerce) que le permite a los usuarios navegar por diferentes categorías de productos, visualizar ofertas y experimentar una interfaz de compra fluida .

###  Tecnologías Utilizadas
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos, diseño visual y maquetación.
* **Responsive Design:** Adaptado para que se vea tanto en computadoras como en dispositivos móviles (celulares y tablets).
